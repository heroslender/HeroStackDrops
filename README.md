## Config

```yaml
restringir-itens:
  method: 'WHITELIST'
  itens:
    - 'INK_SACK'
    - 'COAL'
    - 'COBBLESTONE'
    - 'REDSTONE'
holograma:
  ativado: true
  texto: '&7{quantidade}x &e{nome}'
stack-on-spawn: false
raio-de-stack: 5
```

### Placeholders
-   `{quantidade}` - Quantidade de itens no pack
-   `{nome}` - Nome do item
